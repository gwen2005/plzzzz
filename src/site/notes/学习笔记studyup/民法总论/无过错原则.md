# 民第1166条

</div>


**第一千一百六十六条**　行为人造成他人民事权益损害，不论行为人有无过错，法律规定应当承担侵权责任的，依照其规定。 

</div></div>


无过错原则：即便致害人没有过错仍需承担损害赔偿责任的原则，即致害人承担严格责任 #背诵/概念
