# 法效果：减损请求权的效力
1. 没有否定权利的发生与存续：不能导致请求权消灭
	- 在存在抗辩权的情况下，若仍自愿履行，不认为是不当得利
2. 不影响诉权、胜诉权
3. 使被请求人（义务人）取得一项给付拒绝权（反对权）
	- 行使或主张该请求权据，可以合法地阻碍请求权发挥作用（诉讼上或诉讼外）
	- 行使与否由当事人自己主张
