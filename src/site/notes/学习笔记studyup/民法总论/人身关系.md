# 内容
## 人格关系（[[学习笔记studyup/民法总论/人格权\|人格权]]）：以人格利益为客体
## 身份关系（[[学习笔记studyup/民法总论/身份权\|身份权]]）：以身份利益为客体
# 特征
1. 原则上的专属性
	- 不可移转、不可抛弃
		- 自杀：事实上消灭自己的生命，而不是抛弃生命权
		- 忍辱：只是放弃了对名誉权的救济，而不是抛弃了名誉权
2. 无直接经济价值
- 部分例外：人格权上的财产利益
	- 姓名、肖像、声音语言人格利益等可许可他人使用：可能具有非专属性和经济价值
