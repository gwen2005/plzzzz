拘役：剥夺犯罪分子短期人身自由，就近实行强制劳动改造的刑罚方法
# 与行政拘留的关系
行政拘留属于行政处罚方法，尚未构成犯罪
# 与刑事拘留的关系
- 性质
	- 刑事拘留：属于刑诉中的强制措施，不是处罚方法
	- 拘役：刑罚
- 适用机关
	- 拘役：人民法院
	- 刑事拘留：公安机关
- 期限
	- 拘役：1个月以上6个月以下
	- 刑事拘留：最长37天
# 适用对象
罪行较轻需要关押的犯罪分子
# 期限
一般：1个月以上6个月以下（包括6个月）
数罪并罚：最多不超过1年
# 待遇
1. 就近执行
2. 每月回家一到两天
3. 参加劳动酌量发给报酬
